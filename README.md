# Aviation Recovery Data Visualisation

## Overview

A data visualisation project analysing aviation activity using flight data from the OpenSky Network. The project explored trends in global air traffic before, during, and after the COVID-19 pandemic by applying data analysis and visualisation techniques to identify changes in flight activity and aviation recovery patterns.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Data Visualisation
- Statistical Analysis

## Key Features

- Cleaned and processed large-scale aviation datasets for analysis.
- Analysed flight activity trends across different time periods.
- Developed visualisations to illustrate changes in aviation traffic during the COVID-19 pandemic.
- Identified recovery patterns using statistical analysis and exploratory data analysis techniques.
- Presented data-driven insights through clear and informative visualisations.

## Dataset

- OpenSky Network Aviation Dataset
- Historical flight activity data covering periods before, during, and after the COVID-19 pandemic.

## Results

Successfully analysed aviation recovery trends by identifying significant changes in flight activity over time. The project demonstrated how data visualisation can be used to communicate complex aviation trends and support data-driven decision-making.

## Future Improvements

- Develop an interactive dashboard for real-time visualisation.
- Expand the analysis to include regional and airline-specific trends.
- Integrate predictive models to forecast future aviation activity.
- Incorporate additional aviation datasets for deeper comparative analysis.
