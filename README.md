# Aviation-Recovery-Data-Visualisation
A data visualisation project analysing aviation activity using flight data from the OpenSky Network. The project explored trends in air traffic before and after the COVID-19 pandemic, identifying patterns and presenting insights through data visualisation techniques.

Technologies
1. Python
2. Pandas
3. Data Visualisation
4. Statistical Analysis

Key Features
1. Aviation Traffic Analysis
2. Trend Identification
3. Data Cleaning and Processing
4. Interactive Visualisations
5. Insight Generation
